# DMS Url Shortener

Made to impress! It shortens urls, it shows the most popular, it might even shine your shoes!

## Getting Started

1. Make the DB

  ```ruby
    $ rails db:schema:load
  ```

2. Seed the DB

  ```ruby
    $ rails db:seed
  ```

3. Start yer engines!

  ```ruby
    $ rails s
  ```
